# freebsd-gcc-cross

This is a GCC cross-compiler targeting FreeBSD.

## releases

The current release is based on GCC 15 and supports cross-compiling for the `x86`, `x86_64`, `aarch64`, `riscv64` and `powerpc` targets. The toolchain includes support for the C and C++ frontends.

You can download the prebuilt toolchains from the [releases](https://github.com/AmanoTeam/Loki/releases) page.
